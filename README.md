# Displays the motivation level and active Chant/Refrain

Motivation level counter: displays the current Motivation level

Shield Icon: the shield icon grows progressively more intense with higher tiers of Motivation (Tiers 1-3, 4-6, 7-10)

Unique indicator for each Chant: A unique icon appears on the shield when either Chant of Action, Chant of Recuperation, or Chant of Freedom is active

The level counter, shield, and Chant indicators are all separate components and each of them can be Enabled/Disabled separately

# Installation Guide

Simply extract the zip file inside the reffect folder, and the icons and packages folders will be updated with the Paragon package. Refresh packs in-game afterwards.
